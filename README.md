# RubyKafi

![Ruby](ruby.png)

## Usage

1. Install reveal-md: `sudo npm install -g reveal-md`
1. Install dependencies `npm install`
1. Open the presentation: `reveal-md versions.md --theme puzzle`

## Other Presentations
